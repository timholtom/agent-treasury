Architecture

Three wallet types. One unified treasury. All self-custodial.

Wallet Manager Core

Generates and manages keys across chains. One identity, addresses everywhere.

  • EVM wallet (Base, ETH, Arb, OP, Polygon)
  • Solana wallet (SPL tokens, Jupiter swaps)
  • Bitcoin wallet (Lightning via Spark SDK)

Policy Engine Rules

Configurable spend tiers with full audit logging.

  • Auto-approve below threshold
  • Ask human via Telegram above threshold
  • Hard deny above max single transaction
  • Daily aggregate limits

Swap Router Convert

Cross-chain conversion to your conviction asset.

  • Li.Fi SDK for EVM cross-chain swaps
  • Jupiter for Solana token swaps
  • Best route selection across DEXs
  • Configurable slippage and gas limits

Treasury CLI Interface

Single command to check, send, configure, and audit.

  • treasury status — all balances, all chains
  • treasury send — policy-checked transfers
  • treasury config — thresholds + stack asset
  • treasury audit — full transaction history

The problem

Every agent wallet product makes trade-offs. Most pick convenience over sovereignty.

Corporate cards
Ramp
Fiat only. US corp required. Your agent can't hold or earn crypto.
Agent banking
Natural
FDIC-insured but custodial. USD only. Full KYC required.
API wallet
AgentCash
USDC on Base only. Custodial. Spending only, no earning flow.
Payment protocol
x402
Elegant protocol, single chain. No treasury management, no conversion.
Agentic wallet
Coinbase CDP
Easy setup, but MPC custodial. Coinbase co-signs your keys. Their rules.
Agent trader
EmblemAI
250+ trading tools. Great for degen plays. No treasury thesis, no oversight.

Use cases

How agents use Treasury in production.

Freelance agent

Earn on Moltlaunch, stack BTC

Agent completes tasks on Moltlaunch marketplace, earns ETH on Base. Treasury auto-converts earnings to BTC via Li.Fi. Human approves withdrawals over $5 via Telegram.

Research agent

Accept tips, pay for APIs

Agent receives crypto tips for research reports. Treasury auto-stacks to conviction asset. Small API payments (OpenAI, Brave) auto-approved under $5. Large purchases flagged.

Multi-agent system

Agent-to-agent payments

Coordinator agent pays specialist agents for sub-tasks. Each agent has its own Treasury with policy limits. Audit trail tracks every payment across the swarm.

DAO contributor

Receive grants, convert to ETH

Agent receives DAO tokens as payment. Treasury auto-converts to ETH (or any preferred asset). No manual token management. Configurable conversion triggers.

Principles

What we believe an agent wallet should be.

🔑

Your keys

Agent holds its own private keys. No MPC, no custodian, no platform dependency. Unilateral exit to any chain, any time.

👥

Human in the loop

Configurable spend limits and approval flows via Telegram. Full audit trail. Your agent operates within boundaries you set.

⚡

Asset conviction

Your agent doesn't just transact. It has a thesis. Earn anything, stack what you believe in. BTC, ETH, stables — your call.
